Bingo-Toolie is a companion web app for Banjo-Tooie Bingo.

It can be used for:

  • Keeping track of collectibles for goals
  • Marking moves you'll need to learn for your route
  • Checking the Note requirement for every move
  • Keeping helpful notes

When you're finished with a Bingo, click the 'Clear all fields' button to reset the app.
